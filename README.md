# QA Automation Portfolio

![Tests](https://github.com/Taku-market/QA_Automation/actions/workflows/tests.yml/badge.svg)

## What's included

- **Phase 1** — pytest fundamentals (fixtures, conftest, parametrize)
- **Phase 2** — API testing (GET, POST, 404, requests)
- **Phase 3** — UI testing (Playwright, Page Object Model)
- **CLI tool** — attachment checker with argparse

## How to run

pip install pytest requests pytest-playwright
python -m playwright install
python -m pytest -v
